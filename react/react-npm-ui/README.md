# React Npm UI Library

## Installation

```bash
npm install react-npm-ui
```

## Usage

```javascript
import { Button } from "react-npm-ui";

function App() {
  return (
    <div className="App">
      <Button>Click Me</Button>
    </div>
  );
}

export default App;
```
